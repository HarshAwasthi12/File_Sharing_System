# 📁 File Sharing System

> A clean and simple web-based File Sharing System that allows users to upload and download files from any device, anywhere — without needing to be on the same network.

---

## 🌟 Features

- 📤 Upload files of any type (images, PDFs, videos, ZIPs, etc.)
- 📦 Display file name and size before uploading
- 📥 Download shared files easily
- 🚀 Show uploading/downloading progress
- 🌐 Access files globally using Ngrok
- 💻 Responsive and stylish UI for a smooth experience

---

## 🖥️ Tech Stack

| Frontend       | Backend           | Tools           |
|----------------|-------------------|------------------|
| HTML, CSS, JS  | Python (Flask/Streamlit) | Ngrok, OS, Shutil |

---

## 🚀 Getting Started

### ✅ Prerequisites

- Python 3.x
- pip (Python package installer)
- Ngrok (for sharing externally)

---
